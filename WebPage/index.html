An interactive game that tests the player's spontaneity developed with Android Things

Game Description

The game requires the player's attention and agility in order to press the button at the perfect time. It consists of different levels that test the player's speed of reaction which makes it more fun

Source Code

Pre-requisites

  • Raspberry Pi 3
  • breadboard
  • 12 LEDs
  • jump wires
  • resistor
  • button

Demo

The game consists of 12 LEDs. On the right, there are 3 LEDs that turn on and off representing a binary number (from 1 to 7). Next to them there are 7 LEDs which represent a decimal number (from right to left). The leftmost 2 LEDs are validating ones (RED - incorrect, GREEN - correct). The player has to pay attention to the value in binary, convert it into decimal and press the button when he sees the corresponding value. If he got it right, the green LED turns on, otherwise the red one will. The game gets more intereseting with each correct answer as the speed increases gradually..
